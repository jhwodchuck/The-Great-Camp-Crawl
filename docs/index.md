# The Great Camp Crawl

Welcome to the working docs site for The Great Camp Crawl.

This knowledge base is built around **venue-level records** for overnight programs across the United States, Canada, and Mexico.

## What this project tracks

- overnight summer camps
- specialty residential camps
- sports, arts, academic, and music camps
- family camps
- church and faith-based camps and retreats
- college-run pre-college residential programs

## Core inclusion rules

A candidate should only become a final record when it has evidence for:

1. overnight or residential participation
2. a specific physical venue or session location
3. activity within the last 24 months

## Why venue-level files

A single operator may run:
- multiple camps
- multiple campuses
- multiple session venues

This repository keeps one Markdown file per physical venue so pricing, age ranges, dates, and local details can be tracked accurately.

## Priority lenses

The current research bias favors:
- college-run pre-college residential programs
- one-week-plus programs
- maximum breadth before deep optimization
